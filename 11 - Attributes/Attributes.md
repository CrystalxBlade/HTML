# HTML Attributes

## What is an Attribute?

Attributes provide additional information about HTML elements.

Example:

```html
<img src="image.jpg" alt="Image">
```

Here, `src` and `alt` are attributes.

---

## id

- Must be unique.
- Used for CSS, JavaScript, and bookmark links.

```html
<p id="intro">
```

---

## class

- Can be used on multiple elements.
- Mainly used with CSS and JavaScript.

```html
<p class="note">
```

---

## name

- Mostly used in forms.
- Identifies form data when it is submitted.

```html
<input name="username">
```

---

## title

Shows a tooltip on hover.

```html
<p title="Hello">
```

---

## style

Applies inline CSS.

```html
<p style="color:red;">
```
