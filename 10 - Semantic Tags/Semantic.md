# Semantic HTML

## What is Semantic HTML?

Semantic HTML uses meaningful tags that describe the purpose of the content instead of just its appearance.

Example:

```html
<header>
```

is more meaningful than

```html
<div>
```

---

## Semantic Tags

| Tag | Purpose |
|------|---------|
| `<header>` | Header section |
| `<nav>` | Navigation links |
| `<main>` | Main content |
| `<section>` | A section of content |
| `<article>` | Independent content |
| `<aside>` | Sidebar or related content |
| `<figure>` | Image or illustration |
| `<figcaption>` | Caption for a figure |
| `<footer>` | Footer section |

---

## Why Use Semantic HTML?

- Improves code readability.
- Helps search engines (SEO).
- Improves accessibility.
- Makes code easier to maintain.

---

## Non-semantic Tags

- `<div>`
- `<span>`

These tags don't describe the content's meaning.
