# HTML Lists

## Types of Lists

### 1. Unordered List

- Uses `<ul>`
- List items use `<li>`
- Displays bullet points.

### 2. Ordered List

- Uses `<ol>`
- List items use `<li>`
- Displays numbered or lettered lists.

### Ordered List Types

| Type | Output |
|------|--------|
| `1` | 1,2,3 |
| `A` | A,B,C |
| `a` | a,b,c |
| `I` | I,II,III |
| `i` | i,ii,iii |

### 3. Description List

- `<dl>` → Description List
- `<dt>` → Term
- `<dd>` → Description

### Nested List

A list inside another list.

## Tags Covered

- `<ul>`
- `<ol>`
- `<li>`
- `<dl>`
- `<dt>`
- `<dd>`
