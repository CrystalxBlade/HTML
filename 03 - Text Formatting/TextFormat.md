# HTML Text Formatting

## Tags

| Tag | Purpose |
|------|---------|
| `<b>` | Bold text |
| `<strong>` | Important text |
| `<i>` | Italic text |
| `<em>` | Emphasized text |
| `<u>` | Underlined text |
| `<mark>` | Highlighted text |
| `<small>` | Smaller text |
| `<del>` | Deleted text |
| `<sup>` | Superscript |
| `<sub>` | Subscript |

## Notes

- `<strong>` is preferred over `<b>` when the text has importance.
- `<em>` is preferred over `<i>` when the text needs emphasis.
- `<sup>` is commonly used for exponents.
- `<sub>` is commonly used for chemical formulas.
