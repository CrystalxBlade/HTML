# Introduction to HTML

## What is HTML?

HTML stands for HyperText Markup Language.

It is used to create the structure of a webpage.

HTML is **not** a programming language.
It is a markup language.

---

## Basic Structure

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>

</body>
</html>
